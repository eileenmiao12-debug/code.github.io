Teaching: Java Variables

In Java, a variable is a named container that holds a value. Every variable has a type (such as int, double, or String) and a name that you choose. For example: 

int age = 25;
String name = "Alice";
double price = 19.99;

The type tells Java what kind of data the variable can store. An int stores whole numbers, a double stores decimal numbers, and a String stores text. Variable names should be descriptive and follow Java naming rules: they cannot start with a number, cannot contain spaces, and are usually written in camelCase, like userAge or totalPrice.

You can also change the value of a variable after it is declared: 

int score = 0;
score = 10;   // update the value

Understanding variables is the foundation of Java programming. In this lesson, you will practice declaring variables, choosing correct types, and recognizing valid syntax.

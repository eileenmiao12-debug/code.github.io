Teaching: Java Variables

In Java, a variable is a named container that stores a value. Every variable has a type (such as int, double, or String) and a name that you choose. For example: 

int age = 25;
String name = "Alice";
double price = 19.99;

The type tells Java what kind of data the variable can store. An int stores whole numbers, a double stores decimal numbers, and a String stores text. Variable names must follow Java naming rules: they cannot start with a number, cannot contain spaces, and should be written in camelCase.

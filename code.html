Lesson 1: Variables

Java 变量是用来存储数据的容器。例如：

int age = 18;
String name = "Alice";

Lesson 2: Loops

循环可以让代码重复执行：

for (int i = 0; i < 5; i++) {
    System.out.println(i);
}

Lesson 3: Functions

函数是可以重复使用的代码块：

public static int add(int a, int b) {
    return a + b;
}
